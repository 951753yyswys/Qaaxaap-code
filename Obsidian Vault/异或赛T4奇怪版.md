# 小 l 的祈愿序列

## 题目描述

退坑两年半的小 l 回坑了《原神》，抽卡非常不想歪。

他遇到一个运气极好的原神玩家 z 大神，z大神告诉他：“我给你一段全是 $0$ 的序列，你可以把序列分成若干段，每一段都不能为空哦，分完之后，每个位置的数字都变成它所在段的编号。然后把这些数字全都异或起来，我把得到的值就是的抽卡运气分给你。”

小 l 一听，觉得这事儿可太玄学了！他决定找到一种划分方式，让抽卡运气最大。毕竟，谁不想让抽卡运值爆表呢？但他语文不太好，所以得靠你来帮他算出这个最大值。

## 输入格式

**本题输入包含多组数据，表示小明的多次索要运气。**

第一行一个整数 $T$，表示小明要索要多少次运气。对于每组数据：

- 仅一行，一个正整数 $n$，表示序列长度，也就是这次 z 大神给的序列长度。

## 输出格式

对于每组数据，输出一行一个数，表示最大抽卡运气。

## 输入输出样例 #1

### 输入 #1

```
4
2
3
7
8
```

### 输出 #1

```
3
2
7
8
```

## 输入输出样例 #2

### 输入 #2

```
2
500000000000
100000000000000000
```

### 输出 #2

```
549755813887
144115188075855871
```

## 说明/提示

**【样例解释 #1】**  

当 $n=2$ 时，小 l 可以这样分：$[1,2]$，抽卡运气是 $3$ 。

当 $n=3$ 时，分法变成 $[1,1,2]$，抽卡运气是 $2$。

当 $n=7$ 时，最优的分法：$[1,2,2,3,4,4,5]$，抽卡运气 7。

**【数据范围】**

对于所有数据，保证 $1≤T≤10^5$，$1≤n≤10^{18}$。

各测试点特殊限制如下：

| 测试点编号 |   $n≤$    |       特殊性质        |
| :---: | :-------: | :---------------: |
|  $1$  |   $10$    |         无         |
|  $2$  |   $50$    |         无         |
|  $3$  |   $400$   |         无         |
|  $4$  | $10^{18}$ | $n$ 为 $2$ 的非负整数次幂 |
|  $5$  | $10^{18}$ |         无         |
